mysql约束

约束类型

  • 默认约束 DEFAULT
    为字段设置默认值，如果没有明确为字段赋值，则自动赋予默认值
  • 唯一约束 UNIQUE KEY
    保证记录的唯一性，可以为空
  • 外键约束 FOREIGN KEY
    实现一对一或一对多的关系
